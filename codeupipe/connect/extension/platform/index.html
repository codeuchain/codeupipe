The extension bridges this platform to your local machine. It enables native compute, GPU training, and desktop services — all from a browser tab.

  5. Return here — the dashboard will auto-detect the extension

For Developers

Using the CUP Bridge in your page

// Wait for bridge
window.addEventListener('cup-bridge-ready', async () => {
  // Ping the native host
  const pong = await window.cupBridge.ping();
  console.log(pong);  // {result: {action: 'pong', ts: ...}}

  // Check system status
  const status = await window.cupBridge.status();

  // Delegate to spore_runner
  const health = await window.cupBridge.delegate('/health');

  // Provision a capability
  const recipe = await fetch('recipes/dream-training.json').then(r => r.json());
  await window.cupBridge.provision(recipe);
});
